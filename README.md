# Caddy with Route53 DNS Plugin

This repository provides a Docker container for running the Caddy web server with the Route53 DNS plugin integrated.

The primary purpose of this custom build is to facilitate automated HTTPS certificate management using the ACME DNS challenge mechanism via AWS Route53.

*   **Caddy Web Server:** [https://caddyserver.com](https://caddyserver.com)
*   **Route53 DNS Plugin:** [https://github.com/caddy-dns/route53](https://github.com/caddy-dns/route53)
